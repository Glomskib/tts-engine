## The offer (what the DM is selling)

FlashFlow AI turns a raw phone clip into a finished, captioned, TikTok-Shop-ready
video in ~1–3 min/min of footage. The hook for this segment: **"You film it, we
cut it — free on your first 3 videos."** (Confirm the actual intro offer before send.)

## Personalized-opener formula

Each opener is one specific, true observation + one bridge to the offer. Never
generic ("Love your content!"). The formula, per prospect:

```
[Specific thing about THEIR last/best video]  →  [the friction it implies]  →  [FlashFlow removes that friction]  →  [soft ask]
```

Required fields per prospect to generate a real opener:

| Field            | Example                          | Where it's used in the opener            |
|------------------|----------------------------------|------------------------------------------|
| handle           | @creatorhandle                   | greeting                                 |
| name             | first name if known              | greeting                                 |
| follower_count   | 4,200                            | qualifies <10k                           |
| niche            | skincare / EDC / kitchen gadgets | the "I see you do X" line                |
| recent_video     | the whitening-strips demo        | the specific, true observation           |
| friction_signal  | rough cuts / no captions / shaky | the problem FlashFlow solves             |
| product_category | TikTok Shop category they push   | ties to "Shop Demo" mode                 |

## DM template (personalization slots in CAPS)

> Hey NAME — saw your RECENT_VIDEO, the NICHE stuff is genuinely good. Noticed
> FRICTION_SIGNAL though — that's usually the difference between a clip that
> converts on Shop and one that gets scrolled. I built a tool (FlashFlow) that
> takes your raw clip and hands back a captioned, Shop-ready cut in a couple
> minutes. Want me to run your next one through free so you can see the before/after?

Keep it under ~50 words. One emoji max. No links in the first DM (TikTok throttles
link-bearing cold DMs).
